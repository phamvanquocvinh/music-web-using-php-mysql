Sử dụng Xampp để chạy project
Bước 1: Khởi động Xampp, start Apache và MySQL
Bước 2: Tạo database có tên là "db_nhac" và import file db_nhac.sql
Bước 3: Truy cập vào địa chỉ localhost để vào trang web.

Yêu cầu: có kết nối mạng để link jquery, boostrap và các thư viện khác.
